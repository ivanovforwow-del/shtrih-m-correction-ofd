## Основные команды

```bash
# Установка зависимостей
pip install pandas openpyxl requests pywin32

# Подготовка данных из Excel
python prepare_data.py

# Парсинг данных о товарах из чеков
python parse_ofd_receipts.py

# Запуск коррекции чеков (тестовый режим)
python correction_final.py

# Проверка лога
type correction_process.log

# Проверка обработанных чеков
type processed.json
```

## Настройки в correction_final.py

### Основные настройки
- `CSV_FILE = 'list.csv'` - основной CSV-файл с данными чеков
- `ITEMS_FILE = 'receipts_data.csv'` - CSV-файл с данными о товарах
- `LOG_FILE = 'correction_process.log'` - файл лога операций
- `PROCESSED_FILE = 'processed.json'` - файл с обработанными фискальными признаками

### Режим работы
- `MODE = 'test'` или `'prod'` - режим работы ('test' = 1 чек, 'prod' = все чеки)

### Настройки подключения к ККТ
- `COM_PORT = 3` - номер COM-порта
- `BAUD_RATE = 115200` - скорость передачи данных

### Настройки НДС
- `VAT_RATE = 22` - ставка НДС

### Типы товаров и единицы измерения
- `SUBJECT_TYPE_GOOD = 1` - признак обычного товара
- `SUBJECT_TYPE_EXCISE = 2` - признак подакцизного товара
- `MEASURE_UNIT_PIECE = 0` - единица измерения "штуки"
- `MEASURE_UNIT_LITER = 41` - единица измерения "литр"

### Ключевые слова для определения топлива
- `FUEL_KEYWORDS = ['АИ-92', 'АИ-95', 'АИ-98', 'АИ-80', 'ДТ-', 'ДТ ', 'дизель', 'бензин', 'К5', 'газомоторн']`

### Исключение чеков с НДС 10%
- `VAT_10_FP = ['...', '...', ...]` - список фискальных признаков чеков с НДС 10%

## Настройки в prepare_data.py

- `INPUT_FILE = 'Список чеков без НДС Финал.xlsx'` - входной Excel-файл
- `OUTPUT_FILE = 'list.csv'` - выходной CSV-файл
- `SHEET_NAME = 'Лист2'` - имя листа в Excel-файле
- `VAT_COLUMNS = ['НДС', 'Ставка НДС', 'НДС %', 'Ставка']` - возможные названия колонок с НДС

## Настройки в parse_ofd_receipts.py

- `INPUT_FILE = 'Список чеков без НДС Финал.xlsx'` - входной Excel-файл
- `OUTPUT_FILE = 'receipts_data.csv'` - выходной CSV-файл
- `SHEET_NAME = 'Лист2'` - имя листа в Excel-файле
- `REQUEST_DELAY = 0.3` - задержка между HTTP-запросами в секундах

## Форматы файлов

### list.csv
```
summ;type;fiscal_sign
1799.55;0;2444685719
```
- `summ` - сумма чека
- `type` - тип оплаты (0 = безнал, 1 = нал)
- `fiscal_sign` - фискальный признак

### receipts_data.csv
```
row;fp;name;quantity;unit;price;summ;date
3;2444685719;АИ-92-К5;27.0;л;66.65;1799.55;2026-01-01 00:06:00
```
- `row` - номер строки в Excel
- `fp` - фискальный признак
- `name` - наименование товара
- `quantity` - количество
- `unit` - единица измерения
- `price` - цена за единицу
- `summ` - сумма позиции
- `date` - дата и время чека

## Коды ошибок и статусы

### Статусы в логе
- `[INFO]` - информационные сообщения
- `[ERROR]` - ошибки
- `[SUCCESS]` - успешные операции

### Статусы обработки
- `Processed` - чек успешно обработан
- `Skipped (VAT 10%)` - чек пропущен (с НДС 10%)
- `Skipped (mixed payment)` - чек пропущен (смешанная оплата)
- `Error` - ошибка при обработке

## Частые проблемы

### Проблемы с подключением к ККТ
- Проверьте, что `COM_PORT` указан правильно
- Убедитесь, что драйвер ККТ установлен и работает
- Проверьте, что ККТ включена и готова к работе

### Проблемы с Excel-файлами
- Убедитесь, что файл не открыт в Excel
- Проверьте, что формат гиперссылок корректен
- Убедитесь, что имена колонок соответствуют ожидаемым

### Проблемы с зависимостями
- Проверьте, что установлены все зависимости: `pandas`, `openpyxl`, `requests`, `pywin32`
- При проблемах с `pywin32` выполните: `python Scripts/pywin32_postinstall.py -install`

### Проблемы с обработкой данных
- Проверьте, что `list.csv` создан правильно
- Убедитесь, что фискальные признаки содержат только цифры
- Проверьте, что суммы положительны и формат корректен

## Быстрые решения

### Сброс обработанных чеков
```bash
del processed.json
```

### Переключение в тестовый режим
В `correction_final.py` измените:
```python
MODE = 'test'
```

### Изменение порта ККТ
В `correction_final.py` измените:
```python
COM_PORT = [номер_порта]
```

### Проверка подключения к ККТ
Создайте временный файл с:
```python
import win32com.client
drv = win32com.client.Dispatch('AddIn.DrvFR')
drv.ComNumber = 3
drv.BaudRate = 115200
result = drv.Connect()
print("Подключение:", "успешно" if result == 0 else "ошибка")
