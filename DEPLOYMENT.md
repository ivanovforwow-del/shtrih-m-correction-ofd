# Развертывание проекта коррекции чеков ККТ

## Обзор

Данный документ описывает процесс развертывания проекта на новой системе, включая установку зависимостей, настройку окружения и проверку работоспособности.

## Требования к системе

### Операционная система
- Windows 10 или выше (необходима для работы COM-объектов драйвера ККТ)

### Программное обеспечение
- Python 3.8 или выше
- Драйвер ККТ ККТЛаб v.5.20
- Microsoft Excel (для работы с исходными файлами)

### Аппаратные требования
- Наличие COM-порта для подключения ККТ (USB-to-Serial адаптер при необходимости)
- Достаточно свободного места на диске для установки Python и зависимостей (~500MB)

## Подготовка системы

### 1. Установка Python

#### Загрузка и установка
1. Загрузите установщик Python с [официального сайта](https://www.python.org/downloads/)
2. Установите Python 3.8 или выше
3. Убедитесь, что Python и pip добавлены в PATH во время установки

#### Проверка установки
```bash
python --version
pip --version
```

### 2. Установка драйвера ККТ

#### Установка драйвера ККТЛаб v.5.20
1. Загрузите и установите драйвер ККТ ККТЛаб v.5.20
2. Подключите ККТ к компьютеру
3. Убедитесь, что драйвер корректно распознает ККТ
4. Определите используемый COM-порт (обычно COM3, COM4 и т.д.)

#### Проверка подключения
1. Откройте диспетчер устройств Windows
2. Найдите раздел "Порты (COM и LPT)"
3. Убедитесь, что ККТ отображается в списке
4. Запомните номер COM-порта

### 3. Тестирование подключения к ККТ

#### Создание тестового скрипта подключения
Создайте временный файл `test_connection.py`:

```python
import win32com.client

try:
    print("Создание COM-объекта...")
    drv = win32com.client.Dispatch('AddIn.DrvFR')
    print("COM-объект создан")
    
    print("Настройка параметров подключения...")
    drv.ComNumber = 3  # измените на ваш порт
    drv.BaudRate = 115200
    print(f"Параметры подключения установлены: COM{drv.ComNumber}, {drv.BaudRate}")
    
    print("Подключение к ККТ...")
    result = drv.Connect()
    
    if result == 0:
        print("✓ Подключено к ККТ")
        print(f"ECRMode: {drv.ECRMode}")
        
        # Проверка простой операции
        print("Проверка получения версии драйвера...")
        print(f"Driver Version: {drv.DriverVersion}")
        
        print("Отключение от ККТ...")
        drv.Disconnect()
        print("✓ Отключено от ККТ")
    else:
        print(f"✗ Ошибка подключения: {result}")
        print(f"ResultCode: {drv.ResultCode}")
        print(f"ResultCodeDescription: {drv.ResultCodeDescription}")
        
except Exception as e:
    print(f"✗ Ошибка: {e}")
```

#### Запуск теста подключения
```bash
python test_connection.py
```
