# Справочник API проекта коррекции чеков ККТ

## correction_final.py

### Основные функции

#### `correction_refund(kkt, summ, payment_type, fiscal_sign, items, receipt_date)`
Пробивает чек коррекции "Возврат прихода" (отмена ошибочного чека без НДС)

**Параметры:**
- `kkt` - COM-объект драйвера ККТ
- `summ` - сумма чека
- `payment_type` - тип оплаты (0 - безналичные, 1 - наличные)
- `fiscal_sign` - фискальный признак оригинального чека
- `items` - список товаров (массив словарей с полями name, quantity, unit, price, summ)
- `receipt_date` - дата оригинального чека (строка в формате YYYY-MM-DD HH:MM:SS)

**Возвращает:** bool - успешность выполнения операции

#### `correction_sale(kkt, summ, payment_type, items, vat_rate, fiscal_sign, receipt_date)`
Пробивает чек коррекции "Приход" с НДС

**Параметры:**
- `kkt` - COM-объект драйвера ККТ
- `summ` - сумма чека
- `payment_type` - тип оплаты (0 - безналичные, 1 - наличные)
- `items` - список товаров (массив словарей с полями name, quantity, unit, price, summ)
- `vat_rate` - ставка НДС (например, 22)
- `fiscal_sign` - фискальный признак оригинального чека
- `receipt_date` - дата оригинального чека (строка в формате YYYY-MM-DD HH:MM:SS)

**Возвращает:** bool - успешность выполнения операции

#### `process_corrections(data, items_data, dates_data, mode)`
Обрабатывает список чеков коррекции

**Параметры:**
- `data` - массив данных чеков (с полями summ, type, fiscal_sign)
- `items_data` - словарь данных о товарах (ключ - фискальный признак, значение - массив товаров)
- `dates_data` - словарь дат (ключ - фискальный признак, значение - строка даты)
- `mode` - режим работы ('test' или 'prod')

**Возвращает:** tuple (success_count, error_count) - количество успешных и неуспешных операций

#### `connect_kkt()`
Подключается к ККТ через COM-интерфейс

**Возвращает:** COM-объект драйвера ККТ или None в случае ошибки

#### `disconnect_kkt(kkt)`
Отключается от ККТ

**Параметры:**
- `kkt` - COM-объект драйвера ККТ

#### `load_processed()`
Загружает список уже обработанных чеков из файла `processed.json`

**Возвращает:** массив фискальных признаков обработанных чеков

#### `save_processed(fp)`
Сохраняет фискальный признак обработанного чека в файл `processed.json`

**Параметры:**
- `fp` - фискальный признак чека

#### `load_csv_data(csv_file)`
Загружает данные из CSV файла

**Параметры:**
- `csv_file` - путь к CSV файлу

**Возвращает:** массив данных чеков

#### `load_items_data(items_file)`
Загружает данные о товарах из CSV файла

**Параметры:**
- `items_file` - путь к CSV файлу с данными о товарах

**Возвращает:** tuple (items, dates) - словарь товаров и дат

#### `log(message)`
Записывает сообщение в лог и выводит в консоль

**Параметры:**
- `message` - текст сообщения

### Вспомогательные функции

#### `get_subject_type(item_name)`
Определяет признак предмета расчета для товара

**Параметры:**
- `item_name` - название товара

**Возвращает:** int - 1 для обычного товара, 2 для подакцизного товара

#### `get_measure_unit(item_name, item_unit)`
Определяет единицу измерения для товара

**Параметры:**
- `item_name` - название товара
- `item_unit` - единица измерения из данных (л, шт, кг и т.д.)

**Возвращает:** int - код единицы измерения (0 - штуки, 41 - литр и др.)

#### `send_tlv(kkt, tag, value)`
Отправляет TLV-тег через FNSendTLV

**Параметры:**
- `kkt` - COM-объект драйвера ККТ
- `tag` - номер тега
- `value` - значение тега (строка или байты)

**Возвращает:** bool - успешность отправки

#### `send_stlv_tag(kkt, stlv_tag, tags_dict)`
Отправляет STLV-тег (структурированный)

**Параметры:**
- `kkt` - COM-объект драйвера ККТ
- `stlv_tag` - номер родительского STLV-тега
- `tags_dict` - словарь вложенных тегов {tag_number: (tag_type, value)}

**Возвращает:** bool - успешность отправки

## prepare_data.py

#### `extract_fp_from_hyperlink(hyperlink_formula)`
Извлекает фискальный признак из формулы гиперссылки Excel

**Параметры:**
- `hyperlink_formula` - строка с формулой гиперссылки

**Возвращает:** string - фискальный признак или None

#### `find_vat_column(df)`
Находит колонку с НДС в DataFrame

**Параметры:**
- `df` - pandas DataFrame

**Возвращает:** string - название колонки или None

#### `is_vat_10(row, vat_column)`
Проверяет, содержит ли строка НДС 10%

**Параметры:**
- `row` - строка данных
- `vat_column` - название колонки с НДС

**Возвращает:** bool - является ли НДС 10%

#### `prepare_csv()`
Основная функция подготовки CSV из Excel файла

**Возвращает:** None - результат сохраняется в файл

## parse_ofd_receipts.py

#### `extract_receipt_params(hyperlink)`
Извлекает параметры чека из гиперссылки Платформы ОФД

**Параметры:**
- `hyperlink` - строка с гиперссылкой

**Возвращает:** dict - параметры чека (id, date, fp) или None

#### `fetch_receipt_data(params)`
Получает HTML чека с Платформы ОФД

**Параметры:**
- `params` - параметры чека (id, date, fp)

**Возвращает:** string - HTML содержимое чека или None

#### `parse_receipt_items(html_content, fp)`
Парсит HTML чека и извлекает данные о товарах

**Параметры:**
- `html_content` - HTML содержимое чека
- `fp` - фискальный признак

**Возвращает:** array - массив данных о товарах

#### `read_excel_data()`
Читает данные из Excel файла

**Возвращает:** array - массив данных чеков

#### `main()`
Основная функция парсера чеков

**Возвращает:** None - результат сохраняется в файл

## Константы и настройки

### correction_final.py

- `CSV_FILE` - имя файла с основными данными ('list.csv')
- `ITEMS_FILE` - имя файла с данными о товарах ('receipts_data.csv')
- `LOG_FILE` - имя файла лога ('correction_process.log')
- `PROCESSED_FILE` - имя файла с обработанными чеками ('processed.json')
- `MODE` - режим работы ('test' или 'prod')
- `COM_PORT` - номер COM-порта для подключения к ККТ
- `BAUD_RATE` - скорость передачи данных
- `VAT_RATE` - ставка НДС (по умолчанию 22)
- `SUBJECT_TYPE_GOOD` - признак обычного товара (1)
- `SUBJECT_TYPE_EXCISE` - признак подакцизного товара (2)
- `MEASURE_UNIT_PIECE` - единица измерения штуки (0)
- `MEASURE_UNIT_LITER` - единица измерения литр (41)
- `MEASURE_UNIT_GRAM` - единица измерения грамм (10)
- `MEASURE_UNIT_KILOGRAM` - единица измерения килограмм (11)
- `FUEL_KEYWORDS` - ключевые слова для определения топлива
- `VAT_10_FP` - список фискальных признаков с НДС 10%

### prepare_data.py

- `INPUT_FILE` - имя входного Excel файла ('Список чеков без НДС Финал.xlsx')
- `OUTPUT_FILE` - имя выходного CSV файла ('list.csv')
- `SHEET_NAME` - имя листа в Excel файле ('Лист2')
- `VAT_COLUMNS` - возможные названия колонок с НДС

### parse_ofd_receipts.py

- `INPUT_FILE` - имя входного Excel файла ('Список чеков без НДС Финал.xlsx')
- `OUTPUT_FILE` - имя выходного CSV файла ('receipts_data.csv')
- `SHEET_NAME` - имя листа в Excel файле ('Лист2')
- `REQUEST_DELAY` - задержка между HTTP-запросами
