# Сводка настроек проекта коррекции чеков ККТ

## Обзор

Этот документ содержит сводную информацию обо всех настройках проекта, организованных по файлам и категориям.

## correction_final.py

### Основные настройки файлов
- `CSV_FILE` = `'list.csv'` - основной CSV-файл с данными чеков
- `ITEMS_FILE` = `'receipts_data.csv'` - CSV-файл с данными о товарах
- `LOG_FILE` = `'correction_process.log'` - файл лога операций
- `PROCESSED_FILE` = `'processed.json'` - файл с обработанными фискальными признаками

### Режим работы
- `MODE` = `'prod'` - режим работы ('test' или 'prod')

### Настройки подключения к ККТ
- `COM_PORT` = `3` - номер COM-порта
- `BAUD_RATE` = `115200` - скорость передачи данных

### Настройки НДС
- `VAT_RATE` = `22` - ставка НДС

### Типы товаров и единицы измерения
- `SUBJECT_TYPE_GOOD` = `1` - признак обычного товара
- `SUBJECT_TYPE_EXCISE` = `2` - признак подакцизного товара
- `MEASURE_UNIT_PIECE` = `0` - единица измерения "штуки"
- `MEASURE_UNIT_LITER` = `41` - единица измерения "литр"
- `MEASURE_UNIT_GRAM` = `10` - единица измерения "грамм"
- `MEASURE_UNIT_KILOGRAM` = `11` - единица измерения "килограмм"

### Ключевые слова для определения топлива
- `FUEL_KEYWORDS` = `['АИ-92', 'АИ-95', 'АИ-98', 'АИ-80', 'ДТ-', 'ДТ ', 'дизель', 'бензин', 'К5', 'газомоторн']` - ключевые слова для определения топлива

### Исключение чеков с НДС 10%
- `VAT_10_FP` = `['607805230', '3560220730', '572166526', '760931914', '4075689541', '1035906310', '896116170', '1827453878', '2163166593', '1066150790', '2713257049', '1825692796', '2808322941']` - список фискальных признаков чеков с НДС 10%

## prepare_data.py

## Дополнительные файлы данных

### vat10_fp.txt
- Содержит список фискальных признаков чеков с НДС 10% (по одному на строку)

### correction_process.log
- Файл лога с временной меткой для каждой записи

## Рекомендации по настройке

### Режим работы
- Используйте `MODE = 'test'` для проверки функциональности
- Переключитесь на `MODE = 'prod'` для полной обработки

### Подключение к ККТ
- Убедитесь, что `COM_PORT` соответствует реальному порту ККТ
- Обычно `BAUD_RATE = 115200` подходит для большинства ККТ

### Обновление списка чеков с НДС 10%
- Регулярно обновляйте массив `VAT_10_FP` или файл `vat10_fp.txt`
- Это предотвратит обработку чеков, которые не требуют коррекции
